# Impande Artwork & Visual Assets

This directory contains the visual assets that shape the living ecosystem of Impande. 

The aesthetic of Impande is not just a UI skin; it is a profound reflection of the standard's philosophy. We use earth colors, woven textures, and gentle natural motion to ensure the software feels like a sanctuary of human continuity, an African archive.

## Contents
* `africa/`: Contains the canonical map representations used for the interactive navigation drill-down.
  * `impande-africa-map-v1.png`: The original root map.
* `patterns/`: Textures such as woven grass, beadwork motifs, and clay surfaces.
* `icons/`: Symbolic representations (Seed, Baobab, River, Footprints) replacing generic UI icons.

## Philosophy
* **Before there were books, there were voices.** The artwork here supports the oral and communal traditions of history keeping.
* Imagery should always feel respectful, grounded, and dignified. Never flashy or futuristic.
